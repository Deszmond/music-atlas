# Music Atlas

An interactive, node-based system for exploring global music genres and their relationships through a graph-inspired data structure.

## Overview

Music Atlas is a full-stack oriented project in development that models music genres as interconnected nodes, allowing non-linear exploration of cultural and musical relationships.

The system is designed to combine structured data representation with interactive front-end navigation, with future expansion into backend-driven data management.

## Current Stage

The project is in early development:

- Front-end structure built using HTML and CSS
- Core layout for node-based interaction designed
- JavaScript functionality and interactivity not yet implemented
- Backend layer not yet implemented (planned)

## Planned Full-Stack Architecture

The project is being developed toward a full-stack system:

### Frontend

- Dynamic node-based interface
- Graph-inspired visualisation of genre relationships
- Event-driven interactions for exploring connections

### Backend (Planned)

- API layer to manage genre and relationship data
- Structured database for storing nodes and connections
- Dynamic data retrieval for scalable expansion

### Data Model

- Genres represented as nodes
- Relationships represented as edges (graph structure)
- Support for hierarchical and cross-genre connections

## Technical Direction

- Frontend: HTML, CSS (current), JavaScript (in progress)
- Backend: Planned (API-based architecture)
- Data Structure: Graph-based model for genre relationships
- Architecture Goal: Scalable full-stack system for interactive exploration

## Learning Goals
